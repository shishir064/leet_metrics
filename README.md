# LeetMetric

LeetMetric is a lightweight web app that allows users to quickly view their LeetCode statistics by entering a username. It fetches live data from a public LeetCode API and displays the results in a clean dashboard UI.

The project is built using HTML, Tailwind CSS, and JavaScript, with a focus on responsive design and minimal style aesthetics.
